## ⚠️ Важно: Нужно создать PNG изображение

Telegram **не поддерживает SVG** для превью. Нужно создать PNG версию изображения.

### Шаг 1: Конвертация SVG в PNG

1. **Откройте файл `og-image.svg`** в браузере или редакторе
2. **Конвертируйте в PNG** одним из способов:

#### Способ 1: Онлайн конвертер (рекомендуется)
- Откройте: https://cloudconvert.com/svg-to-png
- Загрузите `og-image.svg`
- Установите размер: **1200x630** пикселей
- Конвертируйте и скачайте `og-image.png`

#### Способ 2: Через браузер
1. Откройте `og-image.svg` в браузере
2. Нажмите F12 (DevTools)
3. Найдите элемент `<svg>` в Elements
4. Правой кнопкой → "Capture node screenshot"
5. Сохраните как `og-image.png`

#### Способ 3: Через редактор
- Откройте `og-image.svg` в Figma, Adobe Illustrator или Inkscape
- Экспортируйте как PNG (1200x630)

### Шаг 2: Загрузите PNG на Netlify

1. Добавьте файл `og-image.png` в папку проекта
2. Загрузите на Netlify (через Git или drag & drop)
3. Убедитесь, что файл доступен по адресу:
   `https://keen-nasturtium-dfce04.netlify.app/og-image.png`

### Шаг 3: Обновите кэш Telegram

После загрузки изменений:

1. **Откройте Telegram**
2. **Найдите бота `@WebpageBot`**
3. **Отправьте команду:**
   ```
   /start
   ```
4. **Отправьте URL вашего сайта:**
   ```
   https://keen-nasturtium-dfce04.netlify.app/
   ```
5. Бот обновит кэш, и превью будет отображаться корректно
