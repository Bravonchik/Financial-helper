# Финансовый помощник - Веб-приложение

Современное веб-приложение для управления личными финансами с интуитивным интерфейсом и полным набором функций для отслеживания доходов, расходов и бюджета.

## 🚀 Возможности

- **Дашборд** - Обзор финансов с ключевыми метриками и графиками
- **Транзакции** - Управление всеми финансовыми операциями с фильтрацией и поиском
- **Бюджет** - Планирование бюджета по категориям с визуализацией прогресса
- **Отчёты** - Детальная аналитика доходов и расходов с различными графиками
- **Профиль** - Настройки личной информации и финансовых параметров

## 📦 Установка и запуск

### ⚠️ ВАЖНО: Использование локального сервера обязательно!

Сайт **не будет работать** при простом открытии файла `index.html` через `file://`. Необходимо запустить локальный веб-сервер.

### 🚀 Быстрый старт (Python - рекомендуется)

1. Откройте терминал/командную строку в папке проекта
2. Запустите сервер:
   ```bash
   python -m http.server 8000
   ```
3. Откройте в браузере: **http://localhost:8000**

### 📖 Подробная инструкция

См. файл **[LOCAL_SERVER.md](LOCAL_SERVER.md)** для детальных инструкций по запуску локального сервера различными способами.

### 🔧 Альтернативные способы:

```bash
# Node.js
npx http-server -p 8000

# PHP
php -S localhost:8000
```

**Почему нужен сервер?** Браузеры блокируют некоторые функции JavaScript (localStorage, модули) при открытии файлов напрямую через `file://`.

## 🔧 Функциональность

### Дашборд
- Отображение текущего баланса
- Метрики доходов и расходов
- Цель сбережений с прогрессом
- Круговая диаграмма расходов по категориям
- Линейный график динамики за 6 месяцев

### Транзакции
- Таблица всех транзакций
- Фильтрация по категориям, типу и месяцу
- Поиск по описанию

### Бюджет
- Планирование месячного бюджета
- Отслеживание расходов по категориям
- Визуализация прогресса
- Предупреждения о превышении бюджета
- Добавление и редактирование категорий
- Выбор цвета для категорий

### Отчёты
- Общая статистика за период
- График доходов и расходов по месяцам
- Анализ по категориям
- Сравнение периодов
- Настройка периода отчёта

### Профиль
- Редактирование личной информации
- Загрузка фото профиля
- Настройка валюты
- Настройка начала месяца
