## 🖼️ Конвертация изображения для превью

Для лучшей совместимости рекомендуется конвертировать `og-image.svg` в PNG формат (1200x630 пикселей).

### Онлайн конвертеры:
1. **CloudConvert** - https://cloudconvert.com/svg-to-png
2. **Convertio** - https://convertio.co/svg-png/
3. **Online-Convert** - https://www.online-convert.com/ru/convert/svg-to-png

### Инструкция:
1. Откройте один из конвертеров выше
2. Загрузите файл `og-image.svg`
3. Установите размер: **1200x630** пикселей
4. Конвертируйте и скачайте `og-image.png`
5. Загрузите `og-image.png` в репозиторий на GitHub
6. Обновите ссылки в `index.html` с `.svg` на `.png`

### Альтернатива через браузер:
1. Откройте `og-image.svg` в браузере
2. Нажмите F12 (открыть DevTools)
3. Найдите элемент `<svg>` в Elements
4. Правой кнопкой → Capture node screenshot
5. Сохраните как `og-image.png` (1200x630)
