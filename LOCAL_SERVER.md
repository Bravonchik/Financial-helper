# 🚀 Инструкция по запуску локального сервера

Для корректной работы сайта необходимо запустить локальный веб-сервер. Это связано с тем, что современные браузеры блокируют некоторые функции JavaScript (например, работу с localStorage и загрузку модулей) при открытии файлов напрямую через `file://`.

## 📋 Способы запуска локального сервера

### 1. Python (Рекомендуется - самый простой способ)

#### Python 3.x:
```bash
# Откройте терминал/командную строку в папке с проектом
cd "C:\Users\Пользователь\OneDrive\Рабочий стол\Сайт"

# Запустите сервер
python -m http.server 8000
```

#### Python 2.x (если установлен):
```bash
python -m SimpleHTTPServer 8000
```

После запуска откройте в браузере: **http://localhost:8000**

---

### 2. Node.js (если установлен Node.js)

#### Установка http-server (один раз):
```bash
npm install -g http-server
```

#### Запуск сервера:
```bash
# Перейдите в папку проекта
cd "C:\Users\Пользователь\OneDrive\Рабочий стол\Сайт"

# Запустите сервер
http-server -p 8000
```

После запуска откройте в браузере: **http://localhost:8000**

---

### 3. PHP (если установлен PHP)

```bash
# Перейдите в папку проекта
cd "C:\Users\Пользователь\OneDrive\Рабочий стол\Сайт"

# Запустите сервер
php -S localhost:8000
```

После запуска откройте в браузере: **http://localhost:8000**

---

### 4. VS Code Live Server (для разработчиков)

1. Установите расширение "Live Server" в VS Code
2. Откройте папку проекта в VS Code
3. Правой кнопкой мыши на `index.html` → "Open with Live Server"

---

### 5. XAMPP / OpenServer (для Windows)

1. Установите XAMPP или OpenServer
2. Скопируйте папку проекта в `htdocs` (XAMPP) или `domains` (OpenServer)
3. Запустите сервер через панель управления
4. Откройте в браузере: **http://localhost/Сайт**

---

## ⚠️ Важные замечания

1. **Не закрывайте терминал** - сервер работает пока открыт терминал. Для остановки нажмите `Ctrl+C`

2. **Порт может быть занят** - если порт 8000 занят, используйте другой:
   ```bash
   python -m http.server 8080
   ```

3. **Проверка работы** - после запуска сервера вы должны увидеть в терминале сообщение типа:
   ```
   Serving HTTP on 0.0.0.0 port 8000 (http://0.0.0.0:8000/) ...
   ```

4. **Доступ с других устройств** - если хотите открыть сайт с телефона/планшета в той же сети:
   - Узнайте IP вашего компьютера: `ipconfig` (Windows) или `ifconfig` (Mac/Linux)
   - Используйте: `http://ВАШ_IP:8000`

## 🔧 Быстрая проверка

После запуска сервера:
1. Откройте браузер
2. Перейдите на `http://localhost:8000`

## 🐛 Решение проблем

**Проблема:** "python не является внутренней или внешней командой"
- **Решение:** Установите Python с [python.org](https://www.python.org/downloads/) и отметьте "Add Python to PATH" при установке

**Проблема:** "Порт уже занят"
- **Решение:** Используйте другой порт: `python -m http.server 8080`

**Проблема:** "Данные не сохраняются"
- **Решение:** Убедитесь, что используете `http://localhost:8000`, а не `file:///`

**Проблема:** "Кнопки не работают"
- **Решение:** Откройте консоль браузера (F12) и проверьте ошибки. Убедитесь, что используете локальный сервер.
