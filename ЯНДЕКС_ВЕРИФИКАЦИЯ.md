# 🔐 Как получить код верификации Яндекс

## 📋 Пошаговая инструкция

### Шаг 1: Зайдите в Яндекс.Вебмастер

1. Откройте: https://webmaster.yandex.ru/
2. Войдите в свой Яндекс аккаунт (или создайте новый)

### Шаг 2: Добавьте сайт

1. Нажмите кнопку **"Добавить сайт"** (или **"+"**)
2. Введите URL вашего сайта: `https://financial-helper-you.netlify.app/`
3. Нажмите **"Добавить"**

### Шаг 3: Выберите способ верификации

Яндекс предложит несколько способов верификации. Выберите **"HTML-тег"**:

1. В списке способов верификации найдите **"HTML-тег"**
2. Нажмите на него
3. Скопируйте код, который будет выглядеть примерно так:
   ```html
   <meta name="yandex-verification" content="1234567890abcdef1234567890abcdef">
   ```

### Шаг 4: Скопируйте только значение content

Из кода:
```html
<meta name="yandex-verification" content="1234567890abcdef1234567890abcdef">
```

Скопируйте только часть после `content="` и до `"`:
```
1234567890abcdef1234567890abcdef
```

### Шаг 5: Вставьте в index.html

1. Откройте файл `index.html`
2. Найдите строку:
   ```html
   <meta name="yandex-verification" content="YOUR_YANDEX_VERIFICATION_CODE">
   ```
3. Замените `YOUR_YANDEX_VERIFICATION_CODE` на скопированный код
4. Должно получиться:
   ```html
   <meta name="yandex-verification" content="1234567890abcdef1234567890abcdef">
   ```

### Шаг 6: Загрузите на сайт

1. Загрузите обновлённый `index.html` на Netlify
2. Дождитесь пересборки сайта (1-2 минуты)

### Шаг 7: Подтвердите верификацию

1. Вернитесь в Яндекс.Вебмастер
2. Нажмите кнопку **"Проверить"** или **"Подтвердить"**
3. Яндекс проверит наличие метатега на вашем сайте
4. Если всё правильно, сайт будет подтверждён ✅

---

## 🎯 Альтернативные способы верификации

Если способ с HTML-тегом не подходит, Яндекс также предлагает:

1. **Файл HTML** — загрузить файл на сайт
2. **Метатег в robots.txt** — добавить в robots.txt
3. **DNS-запись** — для доменов с доступом к DNS

Но **HTML-тег** — самый простой способ для большинства случаев.

---

## ⚠️ Важно

- Код верификации уникален для каждого сайта
- Не публикуйте код верификации в открытом доступе
- После верификации код можно оставить в коде — он не мешает работе сайта

---

## 🔍 Где найти код, если уже добавили сайт

Если вы уже добавили сайт, но не завершили верификацию:

1. Зайдите в Яндекс.Вебмастер
2. Найдите ваш сайт в списке
3. Нажмите на него
4. Перейдите в раздел **"Настройки"** → **"Верификация"**
5. Выберите способ **"HTML-тег"**
6. Скопируйте код

---

**Готово! После верификации вы сможете отслеживать индексацию сайта в Яндекс.Вебмастере.** 🎉
